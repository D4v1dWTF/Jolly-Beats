# Jolly Beats - Music Sharing Platform

COMP3810SEF Group Project

## Project Information

**Project Name:** Jolly Beats

**Website URL:** https://jolly-beats.onrender.com

**Description:** A web application where users can upload, share, play music, create playlists, and discuss music with other users.

**Group Info:** 
- Group Number: 50
- Student Name: Wong Tsz Fung, Wan Kai Ho, Cheung Hoi Him, Holly Lei Stephenson

## Features

### User Authentication
- Register with username, email, and password
- Login and logout functionality
- Secure session management

### Music Management
- Upload songs (mp3, wav, ogg, m4a, max 10MB)
- Browse all songs in the World page
- Rate songs uploaded by other users
- Play music directly from the website
- Edit and delete your own songs
- Search songs by title, artist, album, or genre

### Playlists
- Create custom playlists
- Add your own songs to playlists
- Play entire playlists automatically
- Remove songs from playlists
- Delete playlists

### Forum Discussion
- Create posts to discuss music
- Reply to other users' posts
- Like posts from other users
- Delete your own posts

### User Profile
- Change display name
- Upload custom avatar
- Change password
- View achievements

### Achievements System
- Playlist Creator: Create your first playlist
- First Upload: Upload your first song
- Forum Member: Post your first message
- Song Manager: Delete a song
- Content Moderator: Delete a forum post
- Supporter: Like a forum post
- Conversationalist: Reply to a forum post
- Music Collector: Upload 10 songs
- Music Enthusiast: Listen to music for 5 minutes
- Music Critic: Rate a song for the first time
- Five Star Fan: Give a 5 star rating
- Honest Critic: Give a 1 star rating

### Additional Features
- Responsive design for all devices
- Text size adjustment button
- Achievement notifications
- Song rating system with stars

## Technologies Used

- Node.js with Express.js
- MongoDB with Mongoose
- MongoDB GridFS for music file storage
- EJS template engine
- Session based authentication
- RESTful API endpoints
